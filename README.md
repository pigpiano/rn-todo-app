# rn-todo-app

## 폴더 구조

src 폴더를 만들어서 그 안에 App.js, components 폴더 등을 만들어서 작업한다.

## 기능

- 등록: 할 일 항목을 추가하는 기능
- 수정: 완료되지 않은 할 일 항목을 수정하는 기능
- 삭제: 할 일 항목을 삭제하는 기능
- 완료: 할 일 항목의 완료 상태를 관리하는 기능
